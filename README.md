# task-manager
ターミナルで動くタスク管理ツールです。C++で書かれています。

# 機能
| 番号 | 機能 | 説明 |
|------|------|------|
| 1 | タスクを追加 | タイトル・内容・優先度(1-3)・期限を入力してタスクを作成 |
| 2 | タスクを削除 | 番号を指定してタスクを削除 |
| 3 | タスク一覧表示 | すべてのタスクを一覧で表示 |
| 4 | 優先度順に未完了タスク表示 | 未完了のタスクを優先度の高い順に表示 |
| 5 | 期限が近いタスク表示 | 今日以前が期限のタスクを表示 |
| 6 | タスクを完了にする | 番号を指定してタスクを完了状態にする |
| 7 | タスク検索 | キーワードでタスクを検索 |
| 8 | タスク編集 | タイトル・内容・優先度・期限を個別に編集 |
| 9 | 何もしない | タスクを自動保存してプログラムを終了 |

# 必要条件
- C++コンパイラ(g++)
- makeコマンド

# 使用方法
## １.ビルド
以下を実行して、`task-manager`という実行ファイルを作ります。
```bash
make
```
## ２.実行
```bash
./task-manager
```
## ３.メニューが表示される
番号を入力してEnterを押すと各機能が使えます。
```
１　タスクを追加
２　タスクを削除
３　タスク一覧表示
４　優先度順に未完了タスク表示
５　期限が近いタスク表示
６　タスクを完了にする
７　タスク検索
８　タスク編集
９　何もしない
```
## ４.終了
`9` を入力すると終了します。タスクは `tasks.txt` に自動保存され、次回起動時に読み込まれます。
## ５.後片付け
ビルドで作られたファイルを削除したいときは以下を実行します。
```bash
make clean
```
